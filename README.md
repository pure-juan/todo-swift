# Doit
일정 관리를 쉽게 할 수 있는 iOS 어플리케이션 입니다.

개인적으로 공부 및 사용 목적으로 개발하였습니다.

## 사용한 기술
- Xcode 15.4
- Swift 5
- SwiftUI + MVVM
- Realm `10.53.0`
- Target Version `iOS 16`
